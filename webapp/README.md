# CloakCredit-Pool

FHE-based private lending pool on Ethereum Sepolia.

## Technologies

- Vite
- TypeScript  
- React
- RainbowKit
- Tailwind CSS
- Zama FHE

## Setup

```sh
npm install
npm run dev
```

## Smart Contracts

Deploy to Sepolia:
```sh
npx hardhat run scripts/deploy.js --network sepolia
```
